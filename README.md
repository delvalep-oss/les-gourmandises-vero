# Les Gourmandises de Véro

Landing page pour "Les Gourmandises de Véro" - Service de pâtisseries artisanales pour professionnels de la restauration.

## Cible

Cette landing page cible les professionnels de la restauration (restaurants, cafés, traiteurs) dans un rayon de 15km autour de Toulouse qui souhaitent proposer des desserts artisanaux de qualité sans avoir à les produire eux-mêmes.

## Promesse Cash-First

La page propose un "Pack Découverte" à 50€ permettant aux professionnels de tester les produits avant de s'engager dans des formules plus importantes. Cette approche cash-first permet de:

1. Valider l'intérêt du marché rapidement
2. Générer des revenus dès le lancement
3. Obtenir des retours clients précieux pour ajuster l'offre

## Type de CTA

Le CTA principal est de type **pay** - il dirige vers un paiement Stripe pour commander directement le Pack Découverte à 50€.

## Comment lancer en local

1. Clonez ce dépôt
2. Ouvrez `index.html` dans votre navigateur
3. Pour tester l'intégration de paiement, vous devez configurer une variable d'environnement `KOMPOS_LEADS_API_ORIGIN` pointant vers votre serveur API Stripe

## Fonctionnalités

- Design responsive adapté aux mobiles et desktops
- Intégration Stripe Checkout pour les paiements
- Mise en avant des bénéfices pour les professionnels
- Présentation claire des offres et tarifs
